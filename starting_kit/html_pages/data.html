Data

The data is provided by MUSÉUM NATIONAL D’HISTOIRE NATURELLE. The data set contains 5 classes where each class refers to a type of insect or non-insect i.e.

  1. Bee
  2. Wasp
  3. Butterfly
  4. Other Insect
  5. Other
There are more than 210,000 images provided for this competition.

Preprocessing

All images in dataset are preprocessed and are cropped into squared images which are then resized into 128x128 images.

Go to the FILES tab to download the data and the starting kit. The starting kit contains a very small subset of the data in the folder sample_data for debug purposes. To prepare a challenge submission, you need to use the large dataset that you will download separately. For the the large dataset, you do NOT have the labels!
